# Lumina Beauty

Full-Stack Laravel E-Commerce Platform for Cosmetic Products.

## Technologies

- Laravel
- PHP
- Blade
- JavaScript
- HTML
- CSS
- MariaDB/MySQL
- Docker
- DDEV
- Mailpit

## Features

- Authentication and Authorization
- Product Management
- Category Management
- Brand Management
- Shopping Cart
- Favorites
- Checkout
- Order Management
- Admin Dashboard
- Email notifications with Mailpit

## Project Status

This project was developed as a Bachelor's thesis in Computer Science - Software Engineering.
