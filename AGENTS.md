 # Lumina Beauty Project Instructions

## Tech Stack
- Laravel + Blade
- CSS + JavaScript
- MySQL/MariaDB
- No Vue.js or React for now
- Use DDEV later for database/backend

## Design Direction
- Luxury beauty/fragrance storefront
- Inspired by Fragrance Obsession but unique
- Minimal, elegant, premium UI
- Wide spacious layout
- Editorial beauty aesthetic
- Soft neutral/light tones
- Smooth animations
- Product-focused design

## UI Rules
- Keep section typography proportional
- Use consistent premium fonts
- Categories section should be full-width
- Product cards image-focused
- Elegant hover overlays
- Carousel progress line centered in spacing
- Worldwide Brands uses real brand logos in carousel format

## Product Card Style
- Full image cards
- Minimal overlays
- Soft hover darkening
- “See More” on hover
- Sale prices in red
- No heavy borders or crowded UI

## Important
- Do not redesign entire structure unless requested
- Only refine and improve existing UI/UX
- Keep everything balanced and proportional
- Maintain premium luxury storefront feel
